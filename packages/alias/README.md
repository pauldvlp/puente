# @pauldvlp/puente

This is a **scoped alias** for [`puente`](https://www.npmjs.com/package/puente) — the
self-hosted control panel for Cloudflare Tunnels. It installs the exact same CLI;
the two package names are always published at the same version.

```bash
npm install -g @pauldvlp/puente    # then: puente
# or run once:
npx @pauldvlp/puente
```

Prefer the shorter name? Use [`puente`](https://www.npmjs.com/package/puente) instead —
it's identical.

📖 **Full documentation, setup guide and source:**
https://github.com/pauldvlp/puente
